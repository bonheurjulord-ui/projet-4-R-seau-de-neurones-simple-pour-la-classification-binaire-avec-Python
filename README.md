# projet-4-R-seau-de-neurones-simple-pour-la-classification-binaire-avec-Python
Implémentation d’un réseau de neurones simple en Python (NumPy/Keras) pour résoudre un problème de classification binaire (XOR/Iris). Le projet couvre la propagation avant, le calcul de la perte et la rétropropagation. Il illustre l’apprentissage du modèle avec réduction progressive de l’erreur et prédictions finales.
